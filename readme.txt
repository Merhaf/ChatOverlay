자신의 채팅창 링크로 변경하는 법.

1. index.html 를 메모장으로 연다

2. 12번째줄에 있는 http://example.com/ 대신 너의 채팅창 url을 넣는다

3. 저장한다.


채팅창 크기를 바꾸는 법.

1. index.html 를 메모장으로 연다

2. 12번째줄에 있는 width="100%" height="600" 대신 너가 원하는 크기를 넣는다.

3. 저장한다.

4. metadata.txt 를 메모장으로 연다

5. 6번째줄에 있는 Resolution: 200x620 대신 너가 원하는 크기를 넣는다.

6. 저장한다.


채팅창 투명도를 바꾸는 법

1. index.css 를 메모장으로 연다

2. 7번째줄에 있는 opacity: 0.5; 대신 너가 원하는 투명도를 넣는다.

3. 저장한다
